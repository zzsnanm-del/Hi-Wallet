# 地图改动记录

## 概述

用本地静态地图文件（707.yaml + 707.pgm）替代 ROS `/map` 话题，作为所有机器人共用的静态参考地图。不再从 rosbridge 订阅 OccupancyGrid，而是直接从 `public/maps/` 加载渲染。

## 新增文件

### `src/ros_web_gui_app/src/components/layers/StaticMapLayer.tsx`
静态地图渲染层，继承 `BaseLayer`：
- 不订阅任何 ROS 话题（`getMessageType()` 返回 `null`）
- 通过 `THREE.Group` + `PlaneGeometry` + `DataTexture` 渲染
- 使用 `paletteColorCached()` 做 occupancy → RGBA 颜色映射
- 支持 `colorMode`、`alpha`、`height`、`yaw` 等配置项
- **当前默认 yaw = 0**（PGM 像素网格与墙壁平行，无需旋转）

### `src/ros_web_gui_app/src/utils/StaticMapLoader.ts`
从 URL 加载 YAML + PGM 文件，调用 `parsePGMAndYAML()` 返回 `OccupancyGrid` 数据。

### `src/ros_web_gui_app/src/utils/WorldFrameStore.ts`
全局世界坐标系标定存储：
- 每台机器人保存 `{ x, y, theta }` 偏移量
- 持久化到 `localStorage`
- 支持变更监听（`onChange`）

### `src/ros_web_gui_app/public/maps/707.yaml` + `707.pgm`
从本地路径复制的本地地图文件。

## 修改文件

### `src/ros_web_gui_app/src/utils/mapImporter.ts`
- 导出 `parsePGMAndYAML()` 函数（原为私有）
- 新增 YAML 参数解析：`negate`、`occupied_thresh`、`free_thresh`、`mode`
- 按 ROS map_server 约定计算占用概率：`occ_prob = negate ? (1 - ratio) : ratio`
- **移除 Y 轴翻转**：PGM 行序与 THREE.js DataTexture 一致，无需 `height - 1 - y` 翻转

### `src/ros_web_gui_app/src/components/panels/ViewportPanel.tsx`
- 场景初始化时创建 `StaticMapLayer`，异步加载 `/maps/707.yaml`
- Displays 面板新增 "Static Map" 开关
- 添加世界坐标轴渲染（X=红色, Y=绿色, Z=蓝色）
- 添加地面长方向线（±10m）
- 相机初始位置改为 `(0, -8, 7)`（正南方向俯视）
- 移除 `occupancy_grid` 话题订阅
- 新增标定面板：可调整每台机器人的 X/Y/θ 世界偏移，数据持久化到 `WorldFrameStore`

### `src/ros_web_gui_app/src/components/layers/GridLayer.tsx`
- 新增 `setMapDimensions()` 公开方法，地图加载后同步网格尺寸

## 旋转调整历程

1. **初始状态**：无旋转，PGM 像素网格直接对齐世界 XY 轴
2. **尝试 45° 顺时针**（`DEFAULT_MAP_YAW = -Math.PI/4`）：用户反馈 web 坐标轴相对 RViz 顺时针偏移 45°，尝试旋转地图补偿
3. **最终设为 0°**：用户确认 RViz 中 X 轴与地图墙壁平行，PGM 本身已对齐，无需旋转

## Displays 面板层级

```
Static Map     — 本地文件: /maps/707.yaml
Grid           — 仅 go2 显示，尺寸从静态地图同步
PointCloud2    — /utlidar/cloud_deskewed
LaserScan      — /scan
Global Plan    — /plan
Local Plan     — /local_plan
Footprint      — /local_costmap/published_footprint
Front Camera   — /camera/image_raw
RobotModel     — robot_description + TF
TF             — TF frames
```

## 标定功能

- 点击工具栏「标定」按钮打开面板
- 修改 X/Y/θ 后按 Enter 或点击「应用」生效
- 数据自动保存到 `localStorage`（WorldFrameStore）
- 打开标定面板时自动关闭跟随模式
