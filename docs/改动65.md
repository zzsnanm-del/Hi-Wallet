# 改动65 — TB4 专属 Web 面板 UI 修复与布局优化

## 概述

修复 TB4 专属 web 面板的设备状态面板无背景框问题，优化 UI，调整面板布局，修复统计卡片逻辑错误，扩展 RViz 3D 视口尺寸。

---

## 修改的文件

### 1. `src/ros_web_gui_app/src/App.css`

**新增 CSS 样式**：`ManualControlPanel`（卡片容器）、`ControlPanelHeader`/`ControlPanelIcon`/`ControlPanelTitle`（卡片头部）、`ControlButton`（按钮 + hover + active 态）、`ControlButtonRow`（按钮行布局）、`ControlSliderRow`（滑块行）、`ControlHint`（键盘提示文字）。

**原因**：`TB4ControlPanel` 使用了 `ManualControlPanel`、`ControlButton` 等 class，但这些 class 在 CSS 中完全不存在，导致遥控面板没有任何背景框和按钮样式。

**新增代码位置**：App.css 第 261-355 行（`/* Responsive */` 之前）。

```css
.ManualControlPanel {
  background: var(--surface);
  border-radius: var(--radius);
  border: 1px solid var(--border);
  padding: 14px 16px;
  box-shadow: var(--shadow);
  ...
}
.ControlButton {
  min-width: 70px;
  min-height: 36px;
  ...
  border: 1px solid var(--border2);
  background: var(--surface2);
  ...
}
.ControlButton:hover { border-color: var(--accent); color: var(--accent); }
.ControlButton.active { background: var(--accent); color: white; }
...
```

### 3. `src/ros_web_gui_app/src/components/pages/DashboardPage.tsx`

#### 3.1 修复 "全部在线" 统计卡片逻辑

| 之前 | 现在 |
|------|------|
| `delta={totalCount === 0 ? undefined : '▲ 全部在线'}` | `delta={totalCount === 0 ? undefined : onlineCount === totalCount ? '▲ 全部在线' : \`▲ ${onlineCount} 在线\`}` |

之前只要有机器人就显示"全部在线"，即使没有全部在线。现在只有 `onlineCount === totalCount` 时才显示"全部在线"，否则显示实际在线数量。

#### 3.2 扩展 RViz 视口尺寸

| 属性 | 之前 | 现在 |
|------|------|------|
| 侧边栏宽度 | `340px` | `300px` |
| 外层容器 minHeight | `480px` | `620px` |
| 视口面板 minHeight | `480px` | `620px` |

减少侧边栏宽度 40px，增加视口高度 140px，让 RViz 3D 界面更大，减少面板空间浪费。

#### 3.3 TB4 专属侧边栏面板重组

**之前**：
```
📷 CameraVideoPanel
🐢 TurtleBot4Panel
```

**现在**：
```
🐢 TurtleBot4Panel   ← 从位置2移到位置1
🎮 TB4ControlPanel   ← 从设备状态区移到侧边栏
```

`CameraVideoPanel` 从侧边栏移出，`TB4ControlPanel`（遥控面板）从设备状态区移入。

#### 3.4 TB4 设备状态面板精简

**之前**（grid-3 布局，6 个面板）：
```
TB4ControlPanel | RobotCorePanel  | SlamLidarPanel
MotionControlPanel | NetworkPanel | ApiServicesPanel
```

**现在**（单列布局，1 个面板）：
```
CameraVideoPanel
```

删除的面板（仅 TB4 路径，Go2/UAV 不受影响）：
- `RobotCorePanel` — 机器人核心状态
- `SlamLidarPanel` — SLAM / LiDAR
- `MotionControlPanel` — 运动/控制
- `NetworkPanel` — 网络/WebRTC
- `ApiServicesPanel` — API 服务

`CameraVideoPanel` 从侧边栏移到此处，不再使用 grid-3 包裹。
