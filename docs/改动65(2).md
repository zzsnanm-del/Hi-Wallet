# 改动65(2) — RViz 16:9 比例 + TB4 视频流画面嵌入

## 概述

1. RViz 3D 视口改为 **16:9 宽高比**，自动适配不同屏幕宽度
2. TB4 专属面板的"摄像头 / 视频"卡片中嵌入 **MJPEG 实时视频流画面**
3. 遥控面板 D-pad 扁平化重新设计（4键十字，去掉停止键）

---

## 修改的文件

### 1. `src/ros_web_gui_app/src/components/panels/ViewportPanel.css`

**RViz 视口 16:9 比例**：

```css
/* 之前 */
.viewport-panel {
  ...
  min-height: 400px;
}

/* 现在 */
.viewport-panel {
  ...
  aspect-ratio: 16 / 9;   /* 宽高比自动适配 */
}
```

用 `aspect-ratio: 16 / 9` 替代硬编码的 `min-height`，视口宽度随 grid 列自适应，高度按 16:9 自动计算。

### 2. `src/ros_web_gui_app/src/components/pages/DashboardPage.tsx`

#### 2.1 移除 grid 容器的固定 minHeight

```tsx
// 之前
<div style={{ ... minHeight: 620, alignItems: 'stretch' }}>

// 现在
<div style={{ ... alignItems: 'stretch' }}>
```

视口高度由 `aspect-ratio` 控制，不再需要手动指定。

#### 2.2 TB4 CameraVideoPanel 传入视频流 URL

```tsx
// 之前
<CameraVideoPanel connection={activeConn} />

// 现在
<CameraVideoPanel
  connection={activeConn}
  videoSrc={selectedRobot
    ? `http://${selectedRobot.ip}:7654/stream?topic=/oakd/rgb/image_raw&type=mjpeg`
    : undefined
  }
/>
```

使用机器人 IP 动态构造 web_video_server 的 MJPEG 流地址：
- 端口：`7654`
- 端点：`/stream`
- 话题：`/oakd/rgb/image_raw`（OAK-D RGB 相机）
- 编码：`mjpeg`

> **注意**：初版使用了 `/stream_viewer` 端点，但该端点返回的是 **HTML 页面**（`text/html`），不是图像数据。`<img>` 标签拿到 HTML 无法渲染，触发 `onError` 直接 `display: none` 隐藏。验证后确认 `/stream` 端点才返回原始 MJPEG 图像数据（216KB/2秒）。

### 3. `src/ros_web_gui_app/src/components/panels/CameraVideoPanel.tsx`

**新增 `videoSrc` prop + 视频画面渲染**：

```tsx
// Props 新增
interface Props {
  connection: RosbridgeConnection | null;
  videoSrc?: string;   // ← 可选 MJPEG 流 URL
}

// 渲染 — 视频容器 16:9，最小高度 280px
{videoSrc && (
  <div style={{
    marginTop: 12, borderRadius: 8, overflow: 'hidden',
    border: '1px solid var(--border)', background: '#000',
    aspectRatio: '16 / 9', minHeight: 280,
  }}>
    <img
      src={videoSrc}
      alt="Camera stream"
      style={{ width: '100%', height: '100%', objectFit: 'contain', display: 'block' }}
      onError={(e) => { (e.target as HTMLImageElement).style.display = 'none'; }}
    />
  </div>
)}
```

| 属性 | 初版 | 修正后 |
|------|------|--------|
| 视频容器宽高比 | 无 | `aspect-ratio: 16 / 9` |
| 最小高度 | 无（图片原始尺寸） | `min-height: 280px` |
| 图片填充方式 | `height: auto` | `height: 100%; object-fit: contain` |
| 流地址端点 | `/stream_viewer` (HTML) | `/stream` (MJPEG) |

- MJPEG 流通过 `<img>` 标签加载，浏览器自动持续拉取帧
- 视频填满 16:9 容器，在卡片内显著放大展示
- 加载失败时自动隐藏（`onError` → `display: none`）
- 非 TB4 路径不传 `videoSrc`，行为完全不变

### 4. `src/ros_web_gui_app/src/components/TB4ControlPanel.tsx`

**遥控面板 D-pad 重新设计**：

之前（5 键）:
```
        [↑]
  [↶]  [■]  [↷]
        [↓]
```

现在（4 键，扁平化）:
```
        [↑]
  [←]  [↓]  [→]
```

改动：
- 删除停止按钮（■），松开鼠标自动停止
- 左右旋转箭头 `↶` `↷` → 直箭头 `←` `→`
- `↓` 后退移到中间位置
- 键盘快捷键：移除 Q 停止，改为 `WASD 移动`

### 5. `src/ros_web_gui_app/src/App.css`

**D-pad 按钮样式更新**：

| 属性 | 之前 | 现在 |
|------|------|------|
| 网格 | `3×3` (9格, 含 spacer 行) | `3×2` (6格, 紧凑) |
| 最大宽度 | `max-width: 220px` | 无限制（自适应） |
| 按钮圆角 | `10px` | `8px` (更扁平) |
| 按钮高宽比 | 无限制 (瘦长) | `aspect-ratio: 1.4` (宽扁) |
| 边框颜色 | `var(--border2)` | `var(--border)` (更柔和) |
| 字号 | `13px` | `16px` |
| 停止按钮样式 | 红色专用 (`.DpadBtn.stop`) | 已删除 |
